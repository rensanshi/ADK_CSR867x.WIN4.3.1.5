Detailed Description

A utility library to hold broadcast related runtime data so that it can be accessed from within an audio plugin.

It also contains miscellaneous CSB and CELT configuration values that are needed by both kymera and non-kymera audio plugins. At the time of writing there isn't a better place to put them.
