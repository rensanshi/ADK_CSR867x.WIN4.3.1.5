Detailed Description

This message is generated as a result of a call to MapcMasPutMessageRequest().

The confirmation Message for MapcMasPutMessageRequest() and MapcMasPutContinue() operations. The application must call MapcMasPutContinue() if there is more data to push (i.e the received status is mapc_pending ). The final success confirmation message will contain a handle of maximum size of 16 hex digits assigned to the message by the MSE. Each hex digit is represented in one byte.
