Detailed Description

Interface to the SCM library.

This library supports both the broadcaster and receiver roles in a broadcast environment.

A single instance of the SCM library is created for either broadcast or receiver role by calling the appropriate initialisation function.

Typical usage for a broadcast role would be:-

App Library Transport | –> ScmBroadcastInit() –> | | | <– SCM_BROADCAST_INIT_CFM <– | | | –> ScmBroadcastEnable() –> | | | | –> SCM_BROADCAST_TRANSPORT_REGISTER_REQ –> | | | <– ScmBroadcastTransportRegisterCfm() <– | | <– SCM_BROADCAST_ENABLE_CFM <– | | | –> ScmBroadcastMessageReq() –> | | | | –> SCM_BROADCAST_SEGMENT_REQ –> | | | <– ScmBroadcastSegmentCfm() <– | | | –> SCM_BROADCAST_SEGMENT_REQ –> | | | <– ScmBroadcastSegmentCfm() <– | | | –> SCM_BROADCAST_SEGMENT_REQ –> | | | <– ScmBroadcastSegmentCfm() <– | | <– SCM_BROADCAST_MESSAGE_CFM <– | | | –> ScmBroadcastDisable() –> | | | | –> SCM_BROADCAST_TRANSPORT_UNREGISTER_REQ –> | | | <– ScmBroadcastTransportUnRegisterCfm() <– | | <– SCM_BROADCAST_DISABLE_CFM <– | |
