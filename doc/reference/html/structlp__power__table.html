Detailed Description

Power table structure.

Used to define the link policy for an ACL, see ConnectionSetLinkPolicy.

Note
Apart from 'state' and 'time', the other fields of this structure are only used if the lp_power_mode 'state' is 'lp_sniff'. Those fields are as for the HCI_Sniff_Mode command.

Field Documentation

lp_power_mode lp_power_table::state

The power mode, see lp_power_mode .

uint16 lp_power_table::min_interval

Sniff minimum interval. Time = N x 0.625, Range 0x0002 (1.25 msec) to 0xFFFE (40.9), only even values are valid. Only used if lp_power_mode is lp_sniff.

uint16 lp_power_table::max_interval

Sniff maximum interval. Time = N x 0.625, Range 0x0002 (1.25 msec) to 0xFFFE (40.9 sec), only even values are valid. Only used if lp_power_mode is lp_sniff.

uint16 lp_power_table::attempt

Number of Baseband receive slots for sniff attempt. Length = N * 1.25 msec, Range 0x0001 (1.25 msec) to 0x7FFF (40.9 sec) Only used if lp_power_mode is lp_sniff.

uint16 lp_power_table::timeout

Number of Baseband receive slots for sniff timeout. Length = N * 1.25 msec, Range 0x0000 (0 msec) to 0x7FFF (40.9 sec) Only used if lp_power_mode is lp_sniff.

uint16 lp_power_table::time

The time spent in this state of the power table, in seconds. This must be 0 for the last entry in the table.
