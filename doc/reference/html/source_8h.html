bool SourceClose ( Source  source)

Request to close the source.

Parameters
sourceThe source to close
Returns
TRUE if the source could be closed, else FALSE otherwise.

Source stream can be closed only if the requested source stream is not connected to any stream. If the requested stream is connected, then TransformDisconnect or StreamDisconnect should be called before SourceClose.

Some sources, such as RFCOMM connections or the USB hardware, have a lifetime defined by other means, and cannot be closed using this call. And audio ADC and Digital mic sources which have sidetone enabled also cannot be closed untill the sidetone is disabled.

For Pipe streams, a SourceClose(Src) call shall close the source of that stream and the corresponding sink of the other stream, with which this stream shares the common buffer. Also, any transform that exists on these source or sink shall get disconnected.
