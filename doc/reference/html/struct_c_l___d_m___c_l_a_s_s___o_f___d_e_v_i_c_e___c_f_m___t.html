Detailed Description

Message containing the local class of device.

A connection library client may request to read the local class of device. This message is returned in response to such a request. If the local class of device was read successfully then the status field should indicate this and the message will contain a valid device class.

Field Documentation

hci_status CL_DM_CLASS_OF_DEVICE_CFM_T::status

Indicates whether the request to read the local class of device was completed successfully.

uint32 CL_DM_CLASS_OF_DEVICE_CFM_T::dev_class

The local class of device if the request was successful, otherwise invalid.
