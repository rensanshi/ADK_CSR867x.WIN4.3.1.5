uint32 LcdSet ( uint32  mask,
uint32  value 
)

Controls the activation of lcd segments. These segments must have been specified using VM trap PioSetLcdPins().

Parameters
maskEach bit in the mask corresponds to a PIO line, where b0=PIO0. Bits set to 1 will be modified. Bits set to 0 will not be modified.
valueEach bit specifies a PIO's LCD activation: 1=on, 0=off.
Returns
All zeros if successful, otherwise incorrect bits
