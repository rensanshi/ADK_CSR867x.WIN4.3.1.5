Detailed Description

This message is generated as a result of a call to ObexPutRequest().

The application may use ObexObjMapHdrValue() or ObexObjFilterHdrValue() or ObexObjExtractByteHdrValue() API to extract the headers from the Source buffer.

If the status is obex_continue or the last ObexPutrequest() request has the final parameter as FALSE, the application must either call ObexPutRequest() request again to send the rest of the Body header or call ObexAbort() request to abort the PUT operation. In either case, the application must process the source before calling the API.
