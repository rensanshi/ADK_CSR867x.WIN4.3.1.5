Detailed Description

This message is generated for the failure confirmation of a call to AvrcpRegisterNotificationRequest().

This message will be generated if the library is not able to send Register Notification command to the Target device or the Target device is not responding for the Registered event on time. On success, application will not receive this event and it will receive the corresponding event indication message.
