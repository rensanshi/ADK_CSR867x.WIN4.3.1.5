uint8 AGHFP_DIAL_IND_T::number[1]

Pointer to the number to dial. Although the array size is specified a 1, the actual size can be much larger - ie this structure has a variable size, depending on the amount of data. If the client needs access to this data after the message has been destroyed it is the client's responsibility to copy it.
