Detailed Description

Parameters for configuring transmission of broadcast sync train.

Field Documentation

uint16 broadcaster_sync_params::interval_min

Minimum interval between sync train packets in slots. Range: 0x0020-0xFFFE; only even values are valid.

uint16 broadcaster_sync_params::interval_max

Maximum interval between sync train packets in slots. Range: 0x0020-0xFFFE; only even values are valid.

uint32 broadcaster_sync_params::sync_train_timeout

Duration in slots to continue sending the sync train. Configure a value of 0 to continuously send the sync train and not timeout. Range: 0x00000000 or 0x00000002-07FFFFFE; only even values are valid.

uint8 broadcaster_sync_params::service_data

Single octet of application specific data to be included in the sync train. Range: 0x00-0xFF.
