enum adcsel_enum

The possible sources we can take readings from using AdcRequest().

Enumerator
adcsel_aio0 

Test pin AIO0. Read with VM and BCCMD.

adcsel_aio1 

Test pin AIO1. Read with VM and BCCMD.

adcsel_aio2 

Test pin AIO2. Read with VM and BCCMD. Not available on BC7 or BC7-derived chips, e.g. CSR8670

adcsel_aio3 

Test pin AIO3. Read with VM and BCCMD. Not available on BC7 or BC7-derived chips, e.g. CSR8670

adcsel_vref 

The internal reference voltage in the chip. Read with VM and BCCMD.

adcsel_vdd_bat 

Battery voltage at output of the charger (CHG_VBAT). Read with VM and BCCMD.

adcsel_byp_vregin 

Input to bypass regulator (VCHG). Read with VM and BCCMD.

adcsel_vdd_sense 

Battery voltage sense (CHG_VBAT_SENSE). Read with VM and BCCMD.

adcsel_vregen 

VREG_ENABLE voltage. Read with VM and BCCMD.
