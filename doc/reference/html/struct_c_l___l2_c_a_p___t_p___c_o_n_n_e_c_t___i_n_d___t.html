Detailed Description

This message informs the client task of an incoming L2CAP LE COC connection.

This message is used to notify the client task that a remote device is attempting to create an L2CAP LE COC connection to this device. The client task that registered the PSM being connected to will be sent this message. This message should not be ignored, the client must respond even if it wishes to reject the connection attempt.
