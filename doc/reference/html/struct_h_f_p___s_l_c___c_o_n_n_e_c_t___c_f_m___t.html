Detailed Description

This message returns the result of an attempt to establish a Service Level Connection.

This message is sent to the application once a Service Level Connection has been established or the connect attempt has failed. It is sent to inform the application that the Service Level Connection establishment phase has completed. The application will receive this message regardless of whether it initiated the Service Level Connection establishment or whether this was done by the AG.
