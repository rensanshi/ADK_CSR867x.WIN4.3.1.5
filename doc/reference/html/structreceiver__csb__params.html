Detailed Description

Parameters for configuring reception of broadcast data.

Field Documentation

uint16 receiver_csb_params::interval

Interval between CSB packet instances in slots. Range: 0x0002-0xFFFE; only even values are valid.

(CLKNslave - CLK) % 2^38.

Timing accuracy of the master in ppm. Typical values are 20ppm and 250ppm.

uint16 receiver_csb_params::skip

Number of consecutive CSB instants the receiver may skip after a CSB packet is successfully received.
