Detailed Description

This file provides definitions and data types common to the Config Store and Dynamic Config Store libraries. The configuration data will always be stored in the same format regardless of whether it is allocated in const or dynamic memory. Since this data is accessed from multiple places, all the information relating to the data itself is defined within the Config Data library. This library is the single place where all the required types are defined, as well as acting as a utility library for common functions relating to the data.
