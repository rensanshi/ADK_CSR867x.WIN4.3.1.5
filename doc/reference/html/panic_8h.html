Detailed Description

Terminate the application unhappily.

These functions can be used to panic the application, forcing it to terminate abnormally.

uint16 PanicGetAndClearFirmwareLastCode ( bool  clear_panic)

Reads the last panic code and can clear it.

Parameters
clear_panicValue TRUE to clear the last panic code after reading it, otherwise FALSE.
Returns
The last panic code.
Note
The panic code is not cleared at boot-up (due to soft reset or hard reset). Therefore, the VM application can get stale value from power-on or repeated value over multiple power cycles if it was not cleared after boot-up.

VM application can check whether BlueCore has panicked previously first by invoking VmGetResetSource. If the reset source is RESET_SOURCE_FIRMWARE then panic code can be read using this API.
