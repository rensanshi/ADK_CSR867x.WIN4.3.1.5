Detailed Description

Structure used to contains mask or values from all PIOs.

A structure is provided for this as some supported devices have more than 32 PIOs.

Field Documentation

uint32 pio_common_allbits::mask[NUMBER_OF_PIO_BANKS]

Actual mask or value. The accessor functions for this type should be used to access specific bits.

uint32 pio_common_allbits::flags

Flags that may be associated with the PIO state. No meaning is applied to this by the pio_common library.
