Find the index for a named file or directory, relative to the starting directory.

See Also
StreamFileSource, KalimbaLoad
Parameters
startindex of directory in which to start the search
namethe name (possibly including a path) of the item to find
lengththe number of characters in name
Returns
The index of the item searched for, or FILE_NONE if no such item exists.

Leading and trailing directory separators in name are ignored.

start is commonly FILE_ROOT.
