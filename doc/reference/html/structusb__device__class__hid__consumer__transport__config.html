Detailed Description

Structure to pass as params when calling UsbDeviceClassConfigure to set USB_DEVICE_CLASS_CONFIG_HID_CONSUMER_TRANSPORT_DESCRIPTORS. Interface descriptors must be set prior to calling UsbDeviceClassEnumerate if the application wishes to use custom descriptors, otherwise defaults will be used.
