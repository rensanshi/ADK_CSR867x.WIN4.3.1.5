Detailed Description

This message is generate as a result of a call to AvrcpBrowseGetNumberOfItemsRequest().

On success it returns the number of items in the folder and the item list. Item list are defined in section 6.10.2.1, 6.10.2.2 and 6.10.2.3 of Avrcp 1.4 specification for Media player, folder item and media element item respectively. Media element item will contain an attribute value list for the requested attributes in the request.
