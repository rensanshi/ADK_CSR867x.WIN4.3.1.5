Detailed Description

The GATT Server internal structure for the server role.

This structure is visible to the application as it is responsible for managing resource to pass to the "GATT Server" library. The elements of this structure are only modified by the "GATT Server" library. The application SHOULD NOT modify the values at any time as it could lead to undefined behaviour.
