Detailed Description

Configuration metadata.

Useful information about the configuration store and definition file that is required by the wider system, or sent to the Host.

Field Documentation

uint16 config_store_meta_data_t::signature[16]

128bit MD5 hash of the config definition file as an ASCII hex string. Packed as 2 bytes per uint16.

uint32 config_store_meta_data_t::file_size

Size of the compressed file on-device.

uint16 config_store_meta_data_t::filename_length

Equivalent to strlen() of the filename.

uint16 config_store_meta_data_t::filename[1]

On-device path to the compressed config definition. Packed as 2 bytes per uint16.
