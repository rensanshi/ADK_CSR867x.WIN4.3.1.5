What's new in ADK4.3.1?

Voice Assistant support

ADK4.3.1 implements button triggered Voice Assistant (VA) feature that provides support for interacting with a Voice Service Provider (VSP) via a suitable headset using Bluetooth technology. Further details are described in ADK Voice Assistant User Guide
